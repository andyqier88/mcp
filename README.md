# mcp
## mcp-mvp
mcp 最小化可行性server
```
cd mcp-mvp
npm install
npm run build
```

## weather
天气查询mcp
```
cd weather
npm install
npm run build
```
#### mcp 配置
```
{
  "mcpServers": {
    "mcp-mvp": {
      "command": "node",
      "args": [
        "D:\\xx\\xx\\xx\\build\\index.js" // 改成你的路径
      ]
    }
  }
}
```
